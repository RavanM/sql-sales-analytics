# Sales Analytics SQL Project

This project demonstrates how SQL can be used to analyze sales performance across regions, customers, and orders.

## Dataset
The database includes:
- Customers
- Orders
- Sales
- Products
- Regions

## Files
- schema.sql – database structure (tables, keys)
- data.sql – sample data
- analysis.sql – analytical SQL queries

## Business Questions Answered
- What is the total revenue per region?
- Which regions generate the highest sales?
- Who are the top spending customers?
- How many orders does each region generate?

## Skills Demonstrated
- JOINs
- GROUP BY
- Aggregation (SUM, COUNT)
- DISTINCT
- Business-oriented SQL analytics
